Welcome to Wolf Keyboard!

To start using Wolf Keyboard, you need to do a few things:

  1. Enable the Wolf Keyboard input method.

    2. Before an input method can be used, it must be enabled in the Language & Keyboard settings. Press this button to go to the settings page, and check the box labeled "Flow" to enable it.

  2. Select the Wolf Keyboard input method.

    3. Next you must choose which input method to use. Press this button and select Flow from the list.

  3. Read the manual and tutorial.

    4. Before you start using Wolf Keyboard, you may want to read the manual and tutorial. The manual contains complete instructions on how to use Wolf Keyboard. The tutorial introduces you to the Wolf Keyboard layout and gives tips on how to learn it quickly.
