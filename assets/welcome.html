Welcome to Flow!

To start using Flow, you need to do a few things:

  1. Enable the Flow input method.

    2. Before an input method can be used, it must be enabled in the Language & Keyboard settings. Press this button to go to the settings page, and check the box labeled "Flow" to enable it.

  2. Select the Flow input method.

    3. Next you must choose which input method to use. Press this button and select Flow from the list.

  3. Read the manual and tutorial.

    4. Before you start using Flow, you may want to read the manual and tutorial. The manual contains complete instructions on how to use Flow. The tutorial introduces you to the Flow keyboard layout and gives tips on how to learn it quickly.
